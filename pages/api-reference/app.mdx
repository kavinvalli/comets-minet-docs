## App

### Object

| Column    |  Type  |               Description |
| :-------- | :----: | ------------------------: |
| key       | string |       Private key for app |
| hash      | string |       Public hash for app |
| user_hash | string | Public hash for app owner |
| name      | string |       Name of application |

### Endpoints
<br />
#### `GET` /App/?auth=`{user_key}`
##### Parameters
- `{user_key}` string
    
    The key generated for the user.
##### Output
Returns all apps authorized by the user.

#### `GET` /App/owned?auth=`{user_key}`
##### Parameters
- `{user_key}` string
    
    The key generated for the developer.
##### Output
Returns all apps owned by the user.

#### `POST` /App?auth=`{user_key}`
##### Parameters
- `{user_key}` string
    
    The key generated for the developer.
##### Request Body
The request body needs to be in the form of JSON.
```json
{
  "name": "App Name"
}
```
##### Output
- Creates a new app owned by the developer
- Returns the new app.
